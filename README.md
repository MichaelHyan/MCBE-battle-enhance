## MCBE 原版战斗优化

### 使用方式

下载并安装BattleEnhance.mcaddon，适用与MCBE 1.21.110及以上

### 修改内容

未添加新内容，仅对原版进行调整

#### 剑类

所有剑拥有格挡和横扫能力

横扫仿自java版横扫效果，伤害为面板的一半向上取整，有短暂cd

未装备盾牌时下蹲使用格挡，获得抗性3以及饥饿效果

#### 远程

优化准星为x型

大幅减少投射物弹道下坠

少许增加雪球伤害

骷髅掉落箭矢翻倍

持有弓时下蹲缩小视野

持有弩时下蹲大幅缩小视野并附加抗性提升

#### 重锤

从极高处掉落时不再受到掉落伤害

#### 指南针

手持指南针时额外显示方位

#### 鞘翅

装备鞘翅时自动补充烟花火箭

#### 羽毛

手持羽毛时不会误伤

攻击时产生风爆

#### 其他

奖励箱将可额外开出10-15火把

驯服的猫可能会带来更有价值的礼物

大幅降低苦力怕的生成概率，并提高火药爆率
